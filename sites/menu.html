Menu nawigacyjne

Menu nawigacyjne w CSS pozwala tworzyć elementy interfejsu, które umożliwiają poruszanie się po stronie. Możemy budować zarówno menu poziome (np. w nagłówku), jak i menu pionowe (np. w bocznej kolumnie).

W tym przykładzie wykorzystano również układ strony oparty na float, który dzieli stronę na trzy kolumny.

1. Menu poziome w nagłówku

Menu w nagłówku jest tworzone z listy ul, a elementy li ustawiane są obok siebie. 

header ul li {
    display: inline;
}

header a {
    display: inline-block;
    width: 100px;
    text-align: center;
}

2. Menu pionowe

Menu pionowe tworzy się poprzez ustawienie linków jako elementy blokowe. Każdy link zajmuje całą szerokość dostępnej kolumny. 

a {
    display: block;
    width: 100px;
    border: 2px solid black;
    text-align: center;
}

3. Układ strony – 3 kolumny (float)

Cały układ strony został podzielony na trzy części: lewa kolumna, środek oraz prawa kolumna. 

#lewa {
    float: left;
    width: 140px;
}

#srodek {
    float: left;
    width: 700px;
}

#prawa {
    float: right;
    width: 140px;
}

4. Pseudoklasy linków

Linki mogą zmieniać swój wygląd w zależności od stanu: odwiedzony, aktywny lub najechany. 

a:link { background: violet; }
a:visited { background: cyan; }
a:hover { background: lightblue; }
a:active { background: blue; }
