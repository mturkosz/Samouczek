Formularze

Formularze w HTML służą do zbierania danych od użytkownika. Są używane m.in. w logowaniu, rejestracji, ankietach i wyszukiwarkach.

Podstawowa struktura formularza

Każdy formularz zaczyna się od znacznika <form>. 

<form>
    ...
</form>

Pole tekstowe

Najprostsze pole do wpisywania danych to input type="text". 

<input type="text" placeholder="Wpisz imię">

Różne typy pól input

  • text – tekst
  • password – hasło (ukryte znaki)
  • email – adres e-mail
  • number – liczby
<input type="password" placeholder="Hasło">
<input type="email" placeholder="E-mail">
<input type="number" placeholder="Wiek">




Checkbox i radio

Checkbox pozwala zaznaczyć wiele opcji, a radio tylko jedną. 

<input type="checkbox"> HTML
<input type="checkbox"> CSS

<input type="radio" name="opcja"> Tak
<input type="radio" name="opcja"> Nie

Przycisk wysyłania

<input type="submit" value="Wyślij">

Textarea (większe pole tekstowe)

<textarea placeholder="Napisz coś..."></textarea>

Minimalny formularz

<form>
    <input type="text" placeholder="Imię">
    <input type="email" placeholder="E-mail">
    <textarea placeholder="Wiadomość"></textarea>
    <input type="submit" value="Wyślij">
</form>






Podsumowanie

Formularze pozwalają na interakcję użytkownika ze stroną. Dzięki nim można zbierać dane i tworzyć dynamiczne aplikacje internetowe.
