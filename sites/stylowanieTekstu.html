Stylowanie tekstu w CSS

CSS umożliwia pełną kontrolę nad wyglądem tekstu na stronie internetowej. Możemy zmieniać jego wielkość, kolor, odstępy, wyrównanie oraz dekoracje.

Kolor tekstu

Właściwość color odpowiada za kolor tekstu. 

p {
    color: aqua;
}

Przykład tekstu w kolorze aqua.

Wielkość tekstu

Do zmiany rozmiaru tekstu używamy font-size. 

h1 {
    font-size: 40px;
}

p {
    font-size: 2rem;
}

Przykład tekstu 20px

Przykład tekstu 2rem

Wyrównanie tekstu

Właściwość text-align ustawia wyrównanie tekstu. 

p {
    text-align: center;
}

Ten tekst jest wyśrodkowany.

Grubość i styl tekstu

Możemy zmieniać grubość oraz styl tekstu. 

p {
    font-weight: bold;
    font-style: italic;
}

Odstępy między literami i liniami

p {
    letter-spacing: 2px;
    line-height: 1.8;
}

Dekoracja tekstu

Właściwość text-decoration pozwala np. podkreślać tekst. 

a {
    text-decoration: none;
}
