Tabele

Tabele w HTML służą do przedstawiania danych w postaci wierszy i kolumn. Są bardzo przydatne np. do planów lekcji, cenników czy zestawień.

Podstawowe znaczniki tabeli

  • <table> – cała tabela
  • <tr> – wiersz (table row)
  • <th> – nagłówek kolumny
  • <td> – komórka danych

Przykład tabeli

<table>
    <tr>
        <th>Imię</th>
        <th>Wiek</th>
    </tr>
    <tr>
        <td>Jan</td>
        <td>17</td>
    </tr>
    <tr>
        <td>Anna</td>
        <td>16</td>
    </tr>
</table>

Efekt działania

Imię Wiek
Jan 17
Anna 16

Scalanie komórek

Możemy łączyć komórki tabeli:

  • colspan – łączenie kolumn
  • rowspan – łączenie wierszy
<table>
    <tr>
        <th colspan="2">Dane ucznia</th>
    </tr>
    <tr>
        <td>Imię</td>
        <td>Jan</td>
    </tr>
</table>

Efekt

Dane ucznia
Imię Jan

Minimalny CSS (obramowanie)

table {
    border: 2px solid black;
    border-collapse: collapse;
}

th, td {
    border: 1px solid black;
}

Wyjaśnienie CSS

Ten CSS odpowiada tylko za obramowanie tabeli.

  • table – dodaje ramkę całej tabeli
  • th, td – dodaje linie między komórkami
  • border-collapse: collapse – łączy linie, żeby nie były podwójne

Podsumowanie

Tabele pozwalają uporządkować dane w czytelnej formie. Mimo że nie używa się ich do układu strony, nadal są bardzo ważne w HTML.
