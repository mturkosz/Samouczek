Jednostki w CSS

Jednostki w CSS służą do określania rozmiarów elementów takich jak szerokość, wysokość, marginesy czy rozmiar czcionki. Dzielimy je na jednostki bezwzględne oraz względne.

Jednostki bezwzględne

Jednostki bezwzględne mają stałą wartość i nie zależą od ekranu ani rodzica. Najczęściej używa się px.

Jednostka Opis
cm centymetry
mm milimetry
in cale (1in = 96px = 2.54cm)
px piksele (1px = 1/96in)
pt punkty (1pt = 1/72in)
pc pica (12 punktów = 1/6in)

Jednostki względne

Jednostki względne dostosowują się do rozmiaru ekranu lub elementu nadrzędnego. Są podstawą responsywnego projektowania stron.

Jednostka Opis
em względna do rozmiaru czcionki elementu
ex względna do wysokości znaku x w czcionce
ch szerokość znaku "0"
rem względna do html (root), domyślnie 16px
vw 1% szerokości okna
vh 1% wysokości okna
vmin mniejsza wartość z vw i vh
vmax większa wartość z vw i vh
% względna do elementu rodzica

Przykład użycia

.box {
    width: 50%;
    height: 200px;
}

.text {
    font-size: 2rem;
}

Różnica między jednostkami w praktyce:

box (width: 50%)

box (width: 200px)

tekst (font-size: 2rem)

To jest tekst w 2rem

tekst (font-size: 1em)

To jest tekst w 1em

W praktyce najczęściej używa się px, %, rem oraz vw/vh.
