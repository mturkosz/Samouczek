Listy

Listy służą do prezentowania informacji w uporządkowany i czytelny sposób. W HTML wyróżniamy trzy podstawowe rodzaje list: listy nieuporządkowane, uporządkowane oraz definicji.

Lista nieuporządkowana - <ul>

Lista nieuporządkowana wyświetla elementy z użyciem punktorów. 

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Efekt działania

  • HTML
  • CSS
  • JavaScript

Lista uporządkowana - <ol>

Lista uporządkowana numeruje kolejne elementy. 

<ol>
    <li>Uruchom edytor kodu</li>
    <li>Napisz dokument HTML</li>
    <li>Zapisz plik</li>
</ol>

Efekt działania

  1. Uruchom edytor kodu
  2. Napisz dokument HTML
  3. Zapisz plik

Lista definicji - <dl>

Lista definicji służy do przedstawiania pojęć i ich opisów. Wykorzystuje znaczniki <dl>, <dt> oraz <dd>. 

<dl>
    <dt>HTML</dt>
    <dd>Język znaczników stron internetowych.</dd>

    <dt>CSS</dt>
    <dd>Język odpowiedzialny za wygląd strony.</dd>
</dl>

Efekt działania

HTML
Język znaczników stron internetowych.
CSS
Język odpowiedzialny za wygląd strony.

Listy zagnieżdżone

W HTML można tworzyć listy wewnątrz innych list. 

<ul>
    <li>HTML
        <ul>
            <li>Nagłówki</li>
            <li>Listy</li>
        </ul>
    </li>
</ul>

Efekt działania

  • HTML
    • Nagłówki
    • Listy

Znacznik <li> oznacza pojedynczy element listy. Jest on używany zarówno w listach uporządkowanych (<ol>), jak i nieuporządkowanych (<ul>).

Listy są bardzo często wykorzystywane do tworzenia menu nawigacyjnych, spisów treści, instrukcji krok po kroku oraz zestawień informacji.

Typy numeracji w <ol>

Znacznik <ol> pozwala zmieniać sposób numerowania listy za pomocą atrybutu type.

1. Numeracja domyślna (cyfry)

<ol type="1">
    <li>Jeden</li>
    <li>Dwa</li>
    <li>Trzy</li>
</ol>
  1. Jeden
  2. Dwa
  3. Trzy

2. Małe litery

<ol type="a">
    <li>Jabłko</li>
    <li>Gruszka</li>
    <li>Banan</li>
</ol>
  1. Jabłko
  2. Gruszka
  3. Banan

3. Wielkie litery

<ol type="A">
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ol>
  1. HTML
  2. CSS
  3. JavaScript

4. Cyfry rzymskie (małe)

<ol type="i">
    <li>Pierwszy</li>
    <li>Drugi</li>
    <li>Trzeci</li>
</ol>
  1. Pierwszy
  2. Drugi
  3. Trzeci

5. Cyfry rzymskie (duże)

<ol type="I">
    <li>Start</li>
    <li>Środek</li>
    <li>Koniec</li>
</ol>
  1. Start
  2. Środek
  3. Koniec

Podsumowanie

Atrybut type pozwala zmienić wygląd numeracji, ale nie zmienia struktury listy. Elementy nadal są uporządkowane, tylko wyświetlane w inny sposób.
