Menu rozwijalne

Menu rozwijalne jest jednym z najważniejszych elementów każdej strony internetowej. Pozwala ono na uporządkowanie dużej liczby linków w taki sposób, aby użytkownik nie był przytłoczony wszystkimi opcjami naraz.

Zamiast pokazywać wszystkie podstrony od razu, część z nich jest ukryta i pojawia się dopiero po wykonaniu akcji przez użytkownika, najczęściej po najechaniu kursorem myszy lub kliknięciu.

Dzięki temu strony internetowe stają się bardziej czytelne, estetyczne i wygodne w obsłudze.

Menu rozwijalne – jak to działa?

W HTML menu tworzy się najczęściej za pomocą listy <ul> oraz elementów <li>. Każdy element listy może zawierać kolejną listę, która jest podmenu.

CSS odpowiada za to, aby podmenu było domyślnie ukryte i pojawiało się dopiero po interakcji użytkownika. Najczęściej wykorzystuje się do tego pseudoklasę :hover.

1. Menu poziome (najczęściej spotykane)

Menu poziome znajduje się zwykle na górze strony. Główne opcje są ułożone obok siebie, a po najechaniu na jedną z nich rozwija się lista podopcji. 

nav ul li:hover > ul {
    display: block;
}

nav ul ul {
    display: none;
    position: absolute;
}

Tego typu menu jest bardzo popularne, ponieważ pozwala szybko przechodzić między sekcjami strony bez konieczności przewijania.

2. Menu pionowe

Menu pionowe działa podobnie, jednak elementy ustawione są jeden pod drugim. Po najechaniu na daną opcję pojawia się podmenu przesunięte w bok. 

nav ul ul {
    position: absolute;
    left: 140px;
    display: none;
}

3. Menu rozwijalne bez CSS – details i summary

Nowoczesny HTML pozwala tworzyć menu rozwijalne bez użycia CSS i JavaScript. Służą do tego znaczniki <details> oraz <summary>.

Jest to bardzo wygodne rozwiązanie, ponieważ działa automatycznie po kliknięciu i nie wymaga dodatkowych skryptów ani pseudoklas. 

<details>
    <summary>HTML</summary>

    <ul>
        <li>Struktura</li>
        <li>Listy</li>
        <li>Tabele</li>
    </ul>
</details>

Takie menu jest szczególnie przydatne na urządzeniach mobilnych, ponieważ użytkownik nie musi najeżdżać kursorem myszy – wystarczy kliknięcie.

Zalety menu rozwijalnych

Menu rozwijalne mają wiele zalet i dlatego są tak często stosowane.

  • oszczędzają miejsce na stronie,
  • poprawiają czytelność interfejsu,
  • pozwalają grupować podobne elementy,
  • ułatwiają nawigację na dużych stronach,
  • dobrze działają również na urządzeniach mobilnych.
