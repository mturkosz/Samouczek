Tło

CSS umożliwia tworzenie tła elementów na różne sposoby: jednolite kolory, obrazy oraz gradienty (płynne przejścia kolorów).

1. Kolor tła

Najprostsze tło to jednolity kolor ustawiany przez background-color. 

div {
    background-color: lightgray;
}
Przykład tła w kolorze jasnoszarym

2. Obraz jako tło

Możemy ustawić obraz jako tło elementu używając background-image. 

div {
    background-image: url("obraz.jpg");
    background-repeat: no-repeat;
    background-size: cover;
}
Tło z obrazem

3. Właściwości tła

  • background-repeat – czy obraz się powtarza
  • background-position – pozycja obrazu
  • background-size – rozmiar obrazu
div {
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}
Tło wyśrodkowane

4. Gradient liniowy

Gradient liniowy to płynne przejście kolorów w linii prostej. 

div {
    background: linear-gradient(to right, red, blue);
}
Linear gradient

5. Gradient radialny

Gradient radialny rozchodzi się od środka elementu. 

div {
    background: radial-gradient(circle, yellow, green);
}
Radial gradient

6. Skrócony zapis tła

Wszystkie właściwości tła można zapisać w jednej linijce. 

div {
    background: url("tlo.jpg") no-repeat center/cover;
}
