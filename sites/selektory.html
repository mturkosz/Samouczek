Selektory CSS

Selektory CSS służą do wybierania elementów HTML, którym nadajemy style. Dzięki nim możemy precyzyjnie określić, co ma wyglądać w jaki sposób.

1. Selektor uniwersalny

Styluje wszystkie elementy na stronie.

* {
    margin: 0;
    padding: 0;
}

2. Selektor typu

Wybiera wszystkie elementy danego typu.

p {
    color: white;
}

3. Selektor klasy

Może być używany wielokrotnie.

.kolor {
    color: red;
}

Przykład klasy

4. Selektor ID

Unikalny – tylko jeden element na stronie.

#naglowek {
    font-size: 30px;
}

Przykład ID

5. Selektor grupowy

Pozwala stylować kilka elementów jednocześnie.

h1, h3, p {
    color: aqua;
}

6. Selektor potomka

Wybiera elementy znajdujące się wewnątrz innych.

div p {
    color: yellow;
}

Jest potomkiem diva

7. Selektor dziecka

Wybiera tylko bezpośrednie dzieci.

div > p {
    color: lime;
}

8. Selektor atrybutu

Wybiera elementy po atrybucie.

input[type="text"] {
    border: 2px solid black;
    background: white;
}

9. Selektor pseudoklasy

Reaguje na stan elementu.

a:hover {
    color: red;
}
Najedź na mnie

10. Selektor pseudoelementu

Styluje fragment elementu.

p::first-letter {
    font-size: 30px;
}
