Responsywność w CSS

Responsywność to dostosowanie wyglądu strony do różnych rozmiarów ekranu. Strona powinna działać poprawnie zarówno na komputerze, tablecie, jak i telefonie.

Do tworzenia responsywnych stron używa się głównie: media queries (@media) oraz jednostek względnych (%, vw, vh).

1. Media queries – podstawy

Media queries pozwalają zmieniać styl strony w zależności od szerokości ekranu. 

@media (max-width: 768px) {
    body {
        background: lightgray;
    }
}

Zmiana stylu następuje, gdy ekran jest mniejszy niż 768px.

2. Układ elastyczny (flex lub float)

Responsywność często polega na zmianie układu z poziomego na pionowy. 

.kolumny {
    float: left;
    width: 33.33%;
}

3. Responsywne menu

Na dużym ekranie menu może być poziome, a na małym pionowe. 

@media (max-width: 600px) {
    nav a {
        display: block;
        width: 100%;
    }
}

4. Responsywne jednostki

Zamiast px używa się często jednostek względnych. 

h1 {
    font-size: 5vw;
}
