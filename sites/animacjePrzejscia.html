Animacje i przejścia CSS

Animacje w CSS pozwalają na tworzenie dynamicznych efektów wizualnych. Wyróżniamy przejścia (transition) oraz animacje (animation).

1. Transition – podstawy

Transition pozwala płynnie zmieniać właściwości elementu.

.box {
    background: red;
    width: 100px;
    height: 100px;
    transition: 0.3s;
}

.box:hover {
    background: blue;
}
HOVER

2. Transition – kilka właściwości

.box {
    background: green;
    transition: background 0.3s, transform 0.3s;
}

.box:hover {
    background: orange;
    transform: scale(1.2);
}
HOVER

3. Transform – przesunięcia

.box:hover {
    transform: translateX(50px);
}
MOVE

4. Transform – obrót

.box:hover {
    transform: rotate(20deg);
}
ROTATE

5. Transform – skala

.box:hover {
    transform: scale(1.3);
}
SCALE

6. Podstawowa animacja

@keyframes ruch {
    from { transform: translateX(0); }
    to { transform: translateX(200px); }
}

.box {
    animation: ruch 2s infinite;
}

7. Animacja z większą ilością etapów

@keyframes kolor {
    0% { background: red; }
    50% { background: yellow; }
    100% { background: blue; }
}

8. Animacja – ruch + kolor

@keyframes combo {
    0% { transform: translateX(0); background: red; }
    50% { transform: translateX(150px); background: green; }
    100% { transform: translateX(0); background: blue; }
}

9. Opóźnienie animacji

.box {
    animation: ruch 2s infinite;
    animation-delay: 1s;
}

10. Funkcje czasowe animacji

Funkcja czasowa określa tempo wykonywania animacji. Dzięki niej można sprawić, że animacja będzie poruszać się ze stałą prędkością lub przyspieszać i zwalniać. 

.box {
    animation: ruch 3s infinite linear;
}

.box {
    animation: ruch 3s infinite ease;
}

.box {
    animation: ruch 3s infinite ease-in;
}

.box {
    animation: ruch 3s infinite ease-out;
}

.box {
    animation: ruch 3s infinite ease-in-out;
}
linear
ease
ease-in
ease-out
ease-in-out
  • linear – stała prędkość przez całą animację.
  • ease – wolny start, szybszy środek, wolny koniec.
  • ease-in – animacja rozpoczyna się powoli.
  • ease-out – animacja kończy się powoli.
  • ease-in-out – powolny początek i koniec.

11. Kierunek animacji

Właściwość animation-direction określa, w jaki sposób animacja ma się powtarzać. Może odtwarzać się normalnie, od końca do początku lub zmieniać kierunek po każdym cyklu. 

.normal {
    animation-direction: normal;
}

.reverse {
    animation-direction: reverse;
}

.alternate {
    animation-direction: alternate;
}

.alternate-reverse {
    animation-direction: alternate-reverse;
}
normal
reverse
alternate
alternate-reverse
  • normal – od początku do końca.
  • reverse – od końca do początku.
  • alternate – tam i z powrotem.
  • alternate-reverse – najpierw z powrotem, potem tam.
