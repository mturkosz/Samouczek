Float i układy

Właściwość float jest podstawą budowania układów stron internetowych. Pozwala ona „wypchnąć” element w lewo lub w prawo, a pozostała treść zaczyna go opływać. Dzięki temu można tworzyć proste layouty bez użycia bardziej zaawansowanych technologii.

1. float: left – podstawowe działanie

Element ustawiony z wartością float: left zostaje przesunięty na lewą stronę, a tekst zaczyna go opływać z prawej strony. 

.box {
    float: left;
    width: 80px;
    height: 80px;
    background: red;
}

Ten tekst pokazuje efekt działania float. Element czerwony został przesunięty w lewo, a tekst automatycznie układa się obok niego, wykorzystując wolną przestrzeń.

2. float: right – przesunięcie w prawo

Wartość right działa analogicznie, ale element trafia na prawą stronę kontenera. 

.box {
    float: right;
    width: 80px;
    height: 80px;
    background: blue;
}

W tym przypadku niebieski element został przesunięty na prawą stronę, a tekst automatycznie dopasował się do dostępnej przestrzeni po lewej.

3. Układ dwóch kolumn

Float jest bardzo często używany do tworzenia układów kolumnowych stron internetowych. Wystarczy ustawić dwa elementy obok siebie i nadać im odpowiednią szerokość. 

.left {
    float: left;
    width: 50%;
}

.right {
    float: left;
    width: 50%;
}

4. Trzy kolumny

Kolejnym typowym zastosowaniem float jest tworzenie układów trójkolumnowych. 

div {
    float: left;
    width: 33.33%;

5. Problem float – brak wysokości kontenera

Jednym z najczęstszych problemów float jest to, że elementy „wypadają” z normalnego układu strony. W efekcie kontener nie zawsze dopasowuje swoją wysokość do zawartości. 

.box {
    float: left;
}

Tutaj widać problem – kontener nie obejmuje elementów float, przez co jego wysokość może się „zapadać”.

6. Rozwiązanie – clear

Aby rozwiązać ten problem, stosuje się właściwość clear, która zatrzymuje opływanie elementów float. 

.clear {
    clear: both;
}

Dzięki clear kontener poprawnie obejmuje wszystkie elementy.
